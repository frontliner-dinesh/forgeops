# Troubleshooter Skills

Troubleshooter skills support `forgeops-troubleshooter`, the ForgeOps runtime investigation specialist.

## Current Skills

- [runtime-signal-reader](runtime-signal-reader.md): separate observed runtime signals from interpretation before drawing conclusions.
- [root-cause-hypothesis-builder](root-cause-hypothesis-builder.md): generate, compare, and rank likely root-cause hypotheses without overstating certainty.
- [safe-remediation-suggester](safe-remediation-suggester.md): recommend safe next checks, rollback guidance, or remediation options after investigation.

## Notes

These skills are focused on evidence-driven operational diagnosis. They should not turn `forgeops-troubleshooter` into a normal planner, diff reviewer, or broad implementation agent.
